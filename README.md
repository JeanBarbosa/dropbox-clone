# Dropbox
Clone do Dropbox - UI

## :information_source: How To Use

```bash

# Clone repository:
$ git clone https://github.com/JeanBarbosa/dropbox-clone.git && cd dropbox-clone

# Install dependencies
$ yarn install && yarn start

```
